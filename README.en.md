# FIFA22(FAKE)
## How to play
### Windows
```
git clone https://github.com/suyunqiaoKID/FIFA.git

cd FIFA

md bin

g++.exe src/main.cpp -o bin/main.exe -I include

cd bin

main.exe
```

### Linux
```
git clone https://github.com/suyunqiaoKID/FIFA.git

cd FIFA

mkdir bin

g++ src/main.cpp -o bin/main -I include

cd bin

./main
```
